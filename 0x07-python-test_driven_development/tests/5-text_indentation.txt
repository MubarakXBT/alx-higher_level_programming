
=============================
How To Use 5-text_indentation.py
=============================

This module defines a function that indent text 'text_indentation(text)'

Usage:
======
"text_indentation". prints sqaure using # in given size

::

    >>> indent = __import__('5-text_indentation').text_indentation
    >>> indent("Lorem ipsum dolor sit amet, consectetur adipiscing elit. Quonam modo? Utrum igitur tibi litteram videor an totas paginas commovere? Non autem hoc: igitur ne illud quidem. Fortasse id optimum, sed ubi illud: Plus semper voluptatis? Teneo, inquit, finem illi videri nihil dolere. Transfer idem ad modestiam vel temperantiam, quae est moderatio cupiditatum rationi oboediens. Si id dicis, vicimus. Inde sermone vario sex illa a Dipylo stadia confecimus.")
